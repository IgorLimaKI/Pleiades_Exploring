O que é o Pleiades?

Pleiades é uma comunidade responsável por manter diariamente um “dicionário geográfico” de locais antigos, como templos, fortes, cavernas e outras estruturas consideradas portadoras de valor histórico. Esta comunidade possui um database de tamanho considerável, contendo informações como localização, descrição, nome, período relacionado e conexões de mais de 37 mil locais, a maior parte deles localizado no continente europeu e na porção norte do continente africano.

Devido ao volume de dados, é um desafio considerável traduzir o conteúdo reunido por essa comunidade em uma linguagem visual que seja agradável e interessante para o público comum, como pode ser observado no projeto de Ryan M.Horne de observar os locais e suas conexões espalhados pelo planeta. Portanto, o objetivo desta página é explorar alguns dos aspectos dos dados de Pleiades, fornecer representações visuais de seu conteúdo e permitir que seus leitores possam interagir com as descobertas apresentadas.

No mapa abaixo, cada ponto representa um local catalogado. Você pode obter mais informações sobre cada local clicando sobre os pontos, manipulando o zoom ou arrastando o mapa para os lados para ter melhor noção da localização de cada um.

Dados Utilizados

O conjunto de dados usado para a geração dos gráficos é composto originalmente por 3 arquivos separados, locations, places e names, obtidos na página de downloads da comunidade no dia 08/06/2020 07:10 (BRT). Eles foram filtrados e organizados em dois arquivos finais: um CSV contendo os dados gerais de cada lugar e um JSON contendo as algumas conexões registradas entre eles. Dos mais de 36 mil registros, 29767 foram considerados completos e relevantes o suficiente para serem utilizados como base para o painel interativo e apenas 6534 válidos para se apresentar as conexões.
